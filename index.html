CSS Cheat Sheet

Have you encountered moments when you are in a CSS file, and cannot remember some of the most simple CSS code? If you have, it is a very normal and many experienced developers have these kind of moments. I created a guide that can support developers with each of the listed snippets below that can be selected to be used for projects!

Flexbox Row

Use these three properties to create a Flexbox row layout. 

              .row {
                display: flex;
                flex-direction: row;
                flex-wrap: wrap;
              }

Flexbox Column

This helps creating a Flexbox column layout. 

              .column {
                display: flex;
                flex-direction: column;
              }

CSS Grid Layout

This code creates a 12-column layout using CSS Grid. 

              .grid {
                display: grid;
                width: 100%;
                grid-template-columns:
              repeat(12, 1fr);
              }

Linear Gradients

This creates a background using linear gradient coloring from top to bottom. 

              .linear-gradient-background {
                background-image: linear-gradient (
                  rgba (232, 102, 236, 0.3) 0%,
                  rgba (232, 102, 236, 0.6) 100%
                )
              }

Box Transition Glow

Use transition and box shadows to glow on hover. 

              .code-card .card-holder {
                border radius: 8px;
                transition: all 0.5s ease-in-out;
              }

              .code-card:hover,
              .code.card:hover .card-header {
                rgba (232, 102, 236, 1), 0 0 15px
                rgba (232, 102, 236, 1);
              }

Overlay Card With Glow

This uses position properties and negative margins, which raises elements higher than their original starting point. 

              .card-header {
                position relative;
                margin-top: -20px;
              }
